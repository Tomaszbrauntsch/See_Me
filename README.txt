Usage:
1. Change the SerialPort ("/dev/cm.usbXXX") to the one the arduino is sending info to
2. run the node_connect.js via cd into directory then node node_connect.js
3. On browser type in URL http://localhost:3000/data to get the data and the html file to actually get the response of tts
4. Distance Sensor Statistics appear in the terminal that has node active with the distances in inches
